# 加油站问题

在一条环路上有?N?个加油站，其中第?i?个加油站有汽油?gas[i]?升。

你有一辆油箱容量无限的的汽车，从第 i 个加油站开往第 i+1?个加油站需要消耗汽油?cost[i]?升。你从其中的一个加油站出发，开始时油箱为空。

如果你可以绕环路行驶一周，则返回出发时加油站的编号，否则返回 -1。

说明:?

- 如果题目有解，该答案即为唯一答案。
- 输入数组均为非空数组，且长度相同。
- 输入数组中的元素均为非负数。

示例?1:

输入: 
gas  = [1,2,3,4,5]
cost = [3,4,5,1,2]

输出: 3


示例 2:

输入: 
gas  = [2,3,4]
cost = [3,4,3]

输出: -1

## Solution

暴力是一种方法，复杂度比较高。

这里我们想这样一种情况，假如目前已知：从x出发，到达y时不能继续前进了
- 假设我们用curGas\[j\]表示邮箱里还有多少油，那么curGas = $\sum_{i=x}^j gas[i]-cost[i]$，且curGas>0。
- 当车子到达y时，一定有curGas<0，说明在y这一点$gas[y]<cost[y]$。

因此我们考虑，假如到y的时候不能前进了，那么下一步我们该从哪一点开始出发呢？暴力的方法里很明显从x+1的地方作为起点，但这其实是不必要的，证明如下：
- 对于x到y-1中的任意一点k中，一定有curGas[k\]>=0。
- 如果选取k作为起点，那么你在k点的curGas则为0。
- 而从j到y的过程中，$curGas[k]-\sum_{i=k}^y gas[i]-cost[i] < 0$，因此如果你将curGas[k]换为一个一定不比他大的0，那么到y点一定也会没油

因此我们可以采取y+1作为起点，这样整个过程就变成了一次扫描。当扫描到最后一个位置还不能作为出发点，则返回-1。
