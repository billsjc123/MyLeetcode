# 所有奇数长度子数组的和

给你一个正整数数组?arr?，请你计算所有可能的奇数长度子数组的和。

子数组 定义为原数组中的一个连续子序列。

请你返回 arr?中 所有奇数长度子数组的和 。

示例 1：

输入：arr = [1,4,2,5,3]
输出：58
解释：所有奇数长度子数组和它们的和为：
[1] = 1
[4] = 4
[2] = 2
[5] = 5
[3] = 3
[1,4,2] = 7
[4,2,5] = 11
[2,5,3] = 10
[1,4,2,5,3] = 15
我们将所有值求和得到 1 + 4 + 2 + 5 + 3 + 7 + 11 + 10 + 15 = 58
示例 2：

输入：arr = [1,2]
输出：3
解释：总共只有 2 个长度为奇数的子数组，[1] 和 [2]。它们的和为 3 。
示例 3：

输入：arr = [10,11,12]
输出：66

## Solution
除了暴力解法，本题还可以尝试如下思路：判断每个位置总共被纳入子数组了几次，然后通过sum+=次数*数值实现O(n)的解法。

关键问题是：如何求出每个位置被纳入子数组的次数？

首先题目要求奇数组，考虑一个位置i，在他左边总共可以取left个数字，右边可以取right个数字，那么必须得是如下情况才能保证子数组长度为奇数：
1.  左边取奇数个，右边取奇数个
2.  左边取偶数个，右边取偶数个
次数通过排列组合得知，总共有 左奇*右奇+左偶\*右偶种可能性。

因为在左边取的数字的个数的范围从0~i(不取到全取)，因此左边取奇数个的个数是确定的，为left/2，同理左边取偶数个的个数也是确定的，为(left+1)/2。因此答案易得

```java
class Solution {
    public int sumOddLengthSubarrays(int[] arr) {
        int sum = 0;
        for (int i = 0; i < arr.length; i++) {
            int left = i + 1;
            int right = arr.length - (i + 1) + 1;

            int left_odd = left / 2;
            int left_even = (left + 1) / 2;
            int right_odd = right / 2;
            int right_even = (right + 1) / 2;
            sum += (left_odd * right_odd + left_even * right_even) * arr[i];
        }
        return sum;
    }
}
```
