# Leetcode.998 最大二叉树

题目要求：根据数组的位置确定二叉树，父节点比所有子孙结点要大，并且数组中在父节点左侧的数在该节点的左子树，反之则在右子树。

因为新加的数据是数组的最后一项，因此有如下两条规律：

- val 在最后一个，因此每个小于他的子孙结点都在他左子树
- val 在最后一个，因此只能在每个大于他的祖先节点的右子树

对于根节点来说，如果根节点小于 val，那么直接让 val 的左子树是根节点即可；如果根节点大于 val，那么 val 肯定在根节点的右子树中。我们可以继续对右子树的根节点进行类似的判断，发现是同理。

对于这样的结构，很明显可以使用递归。递归函数的返回值是什么呢？

使用递归，对于根节点的操作要能套用到右子树的根节点。因此，下一个进行操作的参数是右子树的根节点。函数最终要返回根节点 root，因此对于这条分支递归的返回值就是参数 root

```java
if(root.val>val)
{
    root.right = insertIntoMaxTree(root.right, val);
    return root;
}
```

对于另外一条分支，只需重复将根节点修改为 val 的操作即可

```java
else{
    TreeNode tmp = new TreeNode(val);
    tmp.left = root;
    return tmp;
}
```
